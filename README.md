## Ukrainian keyboard layout for Mac OS X

### Installation

1. Copy files \*.keylayout та \*.icns from directory `Ukrainian Unicode` to `~/Library/Keyboard Layouts`
2. Log out / log in
3. Select "Ukrainian Unicode" layout in OS X System preferences
